root权限相当于linux系统的顶级管理员，（windows的administrator那种程度的管理员和root是比不了的）虽说root让你的操作方便了不少，玩起系统来也特爽，可是他也带来不少问题，真可谓：“root一时爽，全家火葬场”。。。

root怎么用？。。
在终端启用root账户(也就是你第一次使用root)： 

sudo passwd root
会回显： 
Enter new UNIX password: (在这输入你为root账户指定的密码）
Retype new UNIX password: (确定你输入的密码）
passwd: password updated successfully
以后，如果再次想获得root权限，只需进行如下的操作： 
su root
Password: (在此输入你上面设置的密码）
如果要再次禁用 root 帐号，那么可以执行: 
sudo passwd -l root
root带来的问题：
使用root用户登录系统后，音量大小无法调节：
root用户登录之后运行： 
pulseaudio --start --log-target=syslog
把‘在终端打开’加到右键菜单： 
sudo apt-get install nautilus-open-terminal
在终端打开文件、文件夹： 
nautilus  /文件夹路径
以root权限访问目录： 
chmod o+r /指定的目录
改回 
chmod o-r /指定的目录
在启动界面增加‘root’权限登录系统： 
sudo gedit /etc/lightdm/lightdm.conf.
在下面增加 
greeter-show-manual-login=true 
allow-guest=false
保存即可。
